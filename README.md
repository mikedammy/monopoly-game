# Monopoly Game 
This is a CLI-based Monopoly game written in Python 

The game is fully functional and CLI/Terminal based.

There may still be some areas for improvements, but you can fork the repo or download the files and improve it.

Claude wrote most of the code, I was the architect while Claude was the engineer.
I got token-rate limited many times until I found a way around preserving tokens and it involved me starting new chats for each upgrade, improvement or bug fix I wanted Claude to solve.
